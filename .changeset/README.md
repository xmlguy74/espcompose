# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the repository](https://github.com/changesets/changesets).

## Adding a changeset

To add a changeset, run `pnpm changeset` in the root of the repository. You will be prompted to select
which packages have changed and what type of semver bump (major, minor, patch) each change requires.

A changeset file will be created in this folder describing the changes. Commit it alongside your code changes.

When the release workflow merges the "Version Packages" PR, all pending changesets are consumed,
package versions are bumped, changelogs are updated, and the packages are published to npm.
